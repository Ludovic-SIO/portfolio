Projet : Application de Gestion du Personnel

Présentation : J’ai réalisé un site de gestion du personnel en PHP avec architecture MVC, utilisant MySQL via phpMyAdmin et Bootstrap pour une interface responsive. Les mots de passe sont sécurisés avec SHA-256 et salage.

Fonctionnalités :
  • Page d’accueil (index.php) redirigeant vers la connexion
  • Système d’authentification : login, inscription, hashage des mots de passe
  • Accueil utilisateur avec accès aux employés, ajout de nouveaux employés, déconnexion
  • Interface responsive avec Bootstrap : navbar, boutons, tableaux
Base de données et sécurité :
  • Gestion sécurisée des sessions utilisateurs
  • Mots de passe hachés et salés (SHA-256)
  • Vérification des mots de passe avec password_verify
