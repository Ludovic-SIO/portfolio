Projet : Site e-commerce Symfony

Télécharger la documentation complète

Ce projet consiste à développer un site e-commerce complet dans le cadre de mon BTS SIO SLAM. L'objectif était de mettre en pratique mes compétences en PHP, Symfony et gestion de base de données MySQL, tout en créant une interface attractive et fonctionnelle pour les utilisateurs.

Fonctionnalités principales :

  • Gestion des produits et catégories
  • Panier et paiement sécurisé
  • Interface d’administration pour ajouter, modifier et supprimer des produits
  • Gestion des utilisateurs et suivi des commandes
  • Filtrage et recherche de produits

Technologies utilisées :

  • Symfony pour le back-end et la gestion du routing
  • PHP et MySQL pour la logique serveur et la base de données
  • HTML, CSS, Bootstrap et JavaScript pour le front-end

Objectifs pédagogiques :

  • Mettre en pratique la programmation orientée objet et MVC
  • Apprendre à travailler avec un framework professionnel comme Symfony
  • Gérer un projet web complet avec base de données et interface utilisateur
